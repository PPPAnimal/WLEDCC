# WLEDCC
WLED Command Center.  Discovers and Controls all WLED, MAGICHOME and LEDFX devices from one screen.  Handles Updates to devices and installs LEDFX.  Record and play scenes/presets saved in WLED and LEDFX.  Add cards to even open your favorite web pages (spotify) or programs (winamp).  Truely a All-in-One Control Center.
